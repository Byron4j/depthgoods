

# JDK8并发包概览

以下信息依据 openjdk8描述：

`java\util\concurrent`  直接包里面包含 59 个类或者接口。

`java\util\concurrent\atomic` 里面包含18个类或者接口。

`java\util\concurrent\locks` 里面包含11个类或者接口。
